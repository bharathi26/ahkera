Ahkera

An implementation of the RestMS message broker protocol for message oriented middleware on the Python Django Web Framework.

Ahkera is a rapid prototyping implementation of the RestMS protocol. RestMS defines RESTful operations on AMQP resource types, dropping AMQPs wire format in favor of HTTP. AMQP itself is a message routing protocol for message oriented middleware (or message broker) used in e.g. service oriented architecture implementations.

Find out more about RestMS at the official site http://www.restms.org
   and at the github project http://github.com/pieterh/restms/tree/master.
Read the RestMS Specification Draft: http://wiki.amqp.org/spec:7
Learn about AMQP: http://www.amqp.org
And what, actually, is REST? See http://en.wikipedia.org/wiki/Representational_State_Transfer

Development on Ahkera just started. You are invited to participate - check the github project page for details. The github project wiki is worth a visit, too.
