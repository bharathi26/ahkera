ahkera by t-lo

An implementation of the RestMS protocol for message oriented middleware on the Python Django Web Framework.

BUZZWORD BINGO: Ahkera is a rapid prototyping implementation of the RestMS protocol. RestMS defines RESTful access to AMQP resource types, dropping AMQPs wire format in favor of HTTP. AMQP itself is a message routing protocol for message oriented middleware (or message broker) used in e.g. service oriented architecture implementations.

Find out more about Restms: http://www.restms.org
Read the Specification Draft: http://wiki.amqp.org/spec:7
Learn about AMQP: http://www.amqp.org

Development on Ahkera just started.
